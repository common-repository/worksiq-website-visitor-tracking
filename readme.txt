For embedding the WorksIQ website visitor tracking code into your WordPress website.

== Description ==
WorksIQ CRM alerts your sales team when known prospects visit your website. 

This plugin makes the changes required to your WordPress website to enable the tracking to take place. 

The plugin has no functionality without an active WorksIQ account. For a free trial of WorksIQ please visit [www.worksiq.com](http://www.worksiq.com)

== Installation ==

1. Upload the entire 'worksiq' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

You will find the 'WorksIQ' settings under the WordPress Settings menu. You must edit your settings prior to use.

The plugin has no functionality without an active WorksIQ account. For a free trial of WorksIQ please visit [www.worksiq.com](http://www.worksiq.com)
